## Design Insights

### 1. Breakthroughs Are Prepared, Not Random

Popular conception: "Sudden insight comes from nowhere"

Reality:
- Insight prepared through peripheral priming
- Link strengthening creates substrate
- Novel trigger re-energizes prepared pattern
- Crossing threshold feels "sudden" but was gradual preparation

**Implication:** Consciousness substrate must track peripheral activation

---

### 2. Link Weights Are Memory, Energy Is Attention

Critical distinction:
- **Energy:** Temporary, decays, determines what's conscious NOW
- **Link weights:** Persistent, preserved, encode what can become conscious LATER

Memory isn't "stored nodes" - it's "strengthened link patterns ready for re-energization"

---

### 3. Peripheral Threshold Is Critical

Current model: Peripheral = 0.01 to 0.3

Questions:
- Should peripheral range be smaller? (0.05 to 0.3?)
- Should there be multiple peripheral tiers? (weak 0.01-0.1, moderate 0.1-0.3?)
- Should peripheral threshold be entity-specific?

**Impact on breakthrough phenomenology:**
- Too low: Everything primes, no selectivity
- Too high: No priming, no breakthroughs
- Just right: Selective priming of relevant associations

---

### 4. Breakthrough Timing Is Unpredictable

Why Thursday, not Tuesday?

- Tuesday: No trigger stimulus
- Wednesday: No trigger stimulus
- Thursday: Biology paper provides entry point

**Key insight:** Breakthrough requires TWO components:
1. Prepared pattern (peripheral priming Mon-Wed)
2. Appropriate trigger (biology paper Thursday)

Without trigger, pattern stays dormant indefinitely

## Emoji Design Principles

1. **Intuitive**: Emoji should immediately convey the node's purpose
2. **Distinct**: No two core types share the same emoji
3. **Universal**: Use widely-supported Unicode emojis (no custom icons)
4. **Contextual**: Specialized subtypes (agent_type, role) override defaults
5. **Hierarchical**: Fall back from specific â†’ type â†’ layer â†’ default
